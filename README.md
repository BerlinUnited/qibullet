# qibullet

__Bullet-based__ python simulation interface for __SoftBank Robotics'__ robots.

## Installation

The following modules are required:
* __numpy__
* __pybullet__

The __qibullet__ module can be installed via pip:
```bash
pip install --user qibullet
```
